# SI 320 Graphic Design - Homework 01: Images Curation

## Course
SI 320 Graphic Design and Communications

## Assignment
Homework 01: Curated Collection

## Project Goal
The goal of this assignment is to create a curated image collection that tells a visual story of a one-day experience at my dream place. The collection includes 8–10 images using both direct and symbolic visuals.

## Dream Place
Paris, France

## Visual Story
My project shows a one-day journey through Paris, including sightseeing, food, architecture, and nightlife. The images are organized first by time sequence and then by shot sequence.

## Deliverables
- Page 1: Submission checklist
- Page 2: 8–10 images with time labels and experience descriptions
- Page 3: Same images reorganized by shot progression, with credits and keywords

## Design Focus
This assignment focuses on:
- Image curation
- Visual storytelling
- Shot types and camera angles
- Rule of thirds
- Color unity
- Image credits and organization

## Tools Used
- Photoshop
- Figma / layout design software
- Unsplash images

## Notes
Images were selected to show a clear Paris experience from morning to night, including landmarks, food, streets, and symbolic visuals.
